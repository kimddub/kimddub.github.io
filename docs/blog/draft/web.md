WEB
============

# http
## GET request encoding
    GET 방식으로 파라미터를 넘기는 경우 URL 인코딩에 의해 서버단에서 디코딩하여 사용.
    URLdecoder 또는 tomcat server.xml을 활용.
## POST request encoding
    JSP 단에서 인코딩 및 디코딩 진행.
